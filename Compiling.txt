This guide is designed to quicken the process of compiling LibSourcey in a linux environment. 
The steps below were written for Ubuntu 11.04 - Natty Narwhal - but should be easy to port to other systems.


1) Install Cmake

  sudo apt-get install cmake cmake-qt-gui


1) Install Poco

  sudo apt-get install libpoco-dev
  
  or
  
  http://sourceforge.net/projects/poco/files/sources/poco-1.4.3/poco-1.4.3p1-all.tar.gz/download?use_mirror=waix


1) Install LibSSL

  sudo apt-get install libssl-dev


3) Install OpenCV & FFmpeg 
  http://www.ozbotz.org/opencv-installation/


4) Build LibSourcey
